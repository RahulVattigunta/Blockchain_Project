# Blockchain Project
This project implements a minimal blockchain with:
- Proof-of-Work mining
- Transaction system
- Wallet with cryptographic signing
- P2P node communication
- Web-based Blockchain Explorer

## Setup Instructions

### Install Dependencies
```sh
npm install
```

### Run Blockchain Node
```sh
node node/server.js
```

### Run Wallet
```sh
node wallet/wallet.js
```

### Run UI
```sh
npx http-server ui
```

Then open `http://localhost:8080` in your browser.
